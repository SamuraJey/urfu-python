нужно модифицировать декоратор reverse_string, что бы он если результат функции - строка, то ее нужно перевернуть (сделать reverse). Иначе вернуть None.
def reverse_string(func):
    """Если результат функции - строка, то ее нужно перевернуть. Иначе вернуть None."""
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        return func(*args, **kwargs)
    return wrapper
